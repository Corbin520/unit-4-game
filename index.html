GemsCollector!

You will be given a random number at the start of the game.

There are four gems at the bottom of the page, each with a random value.

The point of the game is to learn the numbers of each gem by clicking on them. each gem has a set value for the game, however, as each game goes on, the value of the gem will change.

Start by clicking on the gem, it will tell you its value.

The random number below will be your target number to get to. If you can get to the random number spot on, YOU WIN!! Be careful, if you go over that score, you lost.
